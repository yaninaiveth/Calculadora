Áreas Formas Geométricas

Forma Área Descripción
Rectángulo
 b x h b: base, h:altura
Cuadrado
 l² l: lado
Trapecio
 (B + b)/2 x h B: base mayor,
b: base menor,
h: altura
Rombo
 (D x d)/2 D: diagonal mayor,
d: diagonal menor,
l: lado
Polígono regular
 P/a P: perímetro, a: apotema
Círculo
 πr² r: radio
Esfera
 4πr² r: radio

Volúmenes Cuerpos Geométricos

Forma Volumen Descripción
Cubo
 a³ a: arista
Ortoedro
 l x a x h l: largo,
a: ancho,
h: altura
Prisma
 Ab x h Ab: área de la base,
h: altura
Cilindro
 πr² x h r: radio, h: altura
Cono o pirámide
 1/3 Ab x h Ab: área de la base,
h: altura
Esfera
 4/3 πr³ r: radio
